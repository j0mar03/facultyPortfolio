## Quick Coding Rules (for consistency)
- Use `NR` = Not Reported.
- Copy exact metric values with units (`%`, `minutes`, `mean`, `SD`, `p-value`).
- Avoid interpretation in Sections 5-8; put interpretation only in Section 10.
- If claim has no number, tag it as `qualitative claim`.
